#1 Differenzierte Lesetexte generieren

Fach: Deutsch
Telli-Ebene: Prompt
Stufen: Primarstufe
Beschreibung: Sachtext in drei Niveaustufen für eine 2. Klasse generieren lassen.

Prompt:
„Schreibe einen kurzen Sachtext über Igel im Winter für eine 2. Klasse. Maximal 60 Wörter, nur einfache Hauptsätze, Silbentrennung durch Bindestriche markieren. Erstelle denselben Text in einer Version für leseschwache Kinder (40 Wörter, Druckschrift-Stil) und einer Version für lesestarke Kinder (80 Wörter, mit einem Nebensatz pro Absatz).“

#2 Elfchen-Gerüste vorbereiten

Fach: Deutsch
Telli-Ebene: Prompt
Stufen: Primarstufe
Beschreibung: Lyrik-Gerüste mit Lücken zum Thema Herbst, Klasse 3.

Prompt:
„Erstelle fünf Elfchen-Gerüste zum Thema Herbst für eine 3. Klasse. Gib jeweils die Struktur 1-2-3-4-1 vor, aber lass die letzte Zeile als Lücke. Ergänze pro Elfchen drei Wortvorschläge für die fehlende Zeile.“
